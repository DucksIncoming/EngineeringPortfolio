Tabulate: Course Scheduler

Background

Tabulate is a course planner I made to help myself set out my course plan for my undergraduate degrees, and I have published it on my GitHub page. The site is made using an HTML front-end and JavaScript to handle the functionality of the scheduling system. The site features saving your work locally and features an option to export your course data so it can be read by DegreeWorks, the credit-management system my university uses.

Link to project: https://ducksincoming.github.io/tabulate.html
